# Bollr — overleveringspakke

Denne mappen inneholder alt for å fortsette Bollr på PC.

## Filer

- **DESIGNNOTAT.md** — alle beslutninger: konsept, palett, typografi, ikoner, kriterier, og teknisk vei videre (Supabase). **Start her.**
- **bollr-forhandsvisning.html** — fungerende demo av hele appen (åpne i nettleser). Bruker demodata i minnet.
- **icon-512.png / icon-180.png / icon-192.png** — hovedikon (lys-deig-spiral på kardemomme-sort).
- **icon-light-512.png / -180.png / -192.png** — lys ikonvariant (kanel-spiral på lys deig).
- **spiral.png** — transparent spiral (favicon / i-app).
- **\*.svg** — vektorkilder for alle ikoner, så de kan regenereres skarpt.

## Raskeste vei videre

1. Les DESIGNNOTAT.md, del 5 (teknisk vei).
2. Kjøp bollr.no, legg demoen på GitHub Pages.
3. Opprett Supabase-prosjekt, sett opp tabeller + Google-innlogging.
4. Bytt demodata i HTML-fila mot Supabase-kall — UI-en er ferdig.

## Status

Design og forhåndsvisning er ferdig. Det som gjenstår er å gjøre dataene ekte og delte (Supabase) og legge appen på nett.
